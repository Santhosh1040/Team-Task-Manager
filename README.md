<h1 align="center">Team Task Manager</h1>

<p align="center">
A full-stack Team Task Management platform built using React, Node.js, Express, Prisma, and PostgreSQL with role-based access control, secure authentication, analytics dashboard, and responsive modern UI.

<h2>Installation & Setup</h2>

<h3>1️⃣ Clone Repository</h3>

<pre>
git clone https://github.com/Santhosh1040/Team-Task-Manager.git
</pre>

---

<h3>2️⃣ Backend Setup</h3>

<pre>
cd Backend
npm install
</pre>

<h4>Create .env</h4>

<pre>
DATABASE_URL=your_postgresql_database_url
JWT_SECRET=your_secret_key
PORT=5000
</pre>

<h4>Run Backend</h4>

<pre>
npx prisma db push
npm start
</pre>

---

<h3>3️⃣ Frontend Setup</h3>

<pre>
cd Frontend
npm install
</pre>

<h4>Create .env</h4>

<pre>
VITE_API_URL=http://localhost:5000
</pre>

<h4>Run Frontend</h4>

<pre>
npm run dev
</pre>
